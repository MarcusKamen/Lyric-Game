Note: There are some issues with this code on checking guess correctness. Also,
sometimes the game does not update the lyric.

Make sure Node.js is installed.
Open the folder in VS code.

Go to terminal in VS code and run:
cd backend
npm install express
npm install cors
npm install csvtojson
npm install axios
node backend.js

Go to another terminal in VS code and run:
cd frontend
npm install react
npm install axios
npm start
